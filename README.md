## 开发

**软链接**

软链接到本地的 skills 目录

```
ln -s ~/project/of/weiran-tech/agent-skills/skills/nuxt3-qa-analysis ~/.agents/skills/nuxt3-qa-analysis
```

**Claude --add-dir**

启动的时候添加目录

```
claude --add-dir ~/project/of/weiran-tech/agent-skills/skills
```
